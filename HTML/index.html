HTML Tutorial

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages.

  • It defines content, not styling
  • Uses tags to tell the browser how to display content
  • Forms the backbone of every website

Why HTML is Important?

  • Foundation of web development
  • Required for frontend, backend & full-stack roles
  • Works with CSS (styling) and JavaScript (logic)
  • SEO friendly when written correctly

Basic Structure of the HTML

<!DOCTYPE html>
<head>
<html>
<title>My First Web Page</title>
</head>
<!body>
<h1>Hello World</h1>

</body>

<!DOCTYPE html>
  • !DOCTYPE html → Defines HTML5
  • html → Root element

HTML Forms

HTML forms are used to collect information from users such as name, email, password, feedback, and other data. Forms send data to a server using GET or POST methods.

Form Tag

The <form> tag is the container for all form elements. 

<form action="submit.php" method="post">
</form>
  • action → URL where form data is sent
  • method → GET or POST

Input Fields

The <input> tag is used to take different types of user input. 

<input type="text" placeholder="Name">
<input type="email" placeholder="Email">
<input type="password" placeholder="Password">
<input type="number" placeholder="Age">

Common Input Types

  • text
  • email
  • password
  • number
  • date
  • file
  • checkbox
  • radio
  • submit

Labels

The <label> tag improves accessibility and usability of forms. 

<label for="email">Email</label>
<input type="email" id="email">

Textarea

The <textarea> tag is used for multi-line text input. 

<textarea rows="4" cols="30"></textarea>

Select Dropdown

The <select> tag creates a dropdown list. 

<select>
  <option>HTML</option>
  <option>CSS</option>
  <option>JavaScript</option>
</select>

Buttons

Buttons are used to submit or reset form data. 

<button type="submit">Submit</button>
<button type="reset">Reset</button>

Form Validation

HTML provides built-in validation using attributes. 

<input type="email" required>
<input type="text" minlength="3">
<input type="number" min="18">

Complete Form Example

<form action="#" method="post">
  <label>Name</label>
  <input type="text" required>

  <label>Email</label>
  <input type="email" required>

  <label>Message</label>
  <textarea></textarea>

  <button type="submit">Submit</button>
</form>

HTML Tables (Detailed)

HTML tables are used to display data in a structured format using rows and columns. They are commonly used for showing records, reports, and comparison data.

Table Tag

The <table> tag is the main container used to create a table. 

<table>
</table>

Table Row

The <tr> tag defines a row in a table. 

<tr>
</tr>

Table Heading

The <th> tag is used for table headings. It is bold and centered by default. 

<th>Name</th>

Table Data

The <td> tag is used to insert data inside table cells. 

<td>Ali</td>

Basic Table Example

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>22</td>
  </tr>
</table>

Table Header, Body and Footer

HTML provides special tags to group table content. 

<table border="1">
  <thead>
    <tr>
      <th>Course</th>
      <th>Duration</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>HTML</td>
      <td>1 Month</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">End of Table</td>
    </tr>
  </tfoot>
</table>

Colspan and Rowspan

These attributes are used to merge table cells. 

<table border="1">
  <tr>
    <th colspan="2">Student Info</th>
  </tr>
  <tr>
    <td>Name</td>
    <td>Ali</td>
  </tr>
</table>

Table Caption

The <caption> tag gives a title to the table. 

<table border="1">
  <caption>Student Table</caption>
  <tr>
    <th>Name</th>
    <th>Marks</th>
  </tr>
</table>

Table Attributes

  • border → Adds border to table
  • cellpadding → Space inside cells
  • cellspacing → Space between cells
  • width → Table width

Best Practices for Tables

  • Use tables only for tabular data
  • Use <th> for headings
  • Use <caption> for accessibility
  • Avoid tables for page layout

Div, Class, ID, Button & Span (Detailed)

These elements are commonly used to structure web pages, group content, apply styling, and add interactivity.

Div

The <div> tag is a block-level container used to group multiple elements together for layout and styling purposes. 

<div>
  <h3>Section Title</h3>
  <p>This is a section of content.</p>
</div>
  • Block-level element
  • Takes full width
  • Used for layout structure

Class

The class attribute is used to apply the same style to multiple elements. One class can be used many times. 

<div class="card">Card 1</div>
<div class="card">Card 2</div>
  • Reusable
  • Used with CSS and JavaScript
  • Multiple elements can share one class

ID

The id attribute uniquely identifies an element on a page. An id must be used only once. 

<div id="header">Header Section</div>
  • Unique
  • Used for JavaScript targeting
  • Used for anchor links

Class vs ID

  • Class → Can be reused
  • ID → Must be unique

Button

The <button> tag is used to create clickable buttons. Buttons are commonly used in forms and user interactions. 

<button>Click Me</button>
<button type="submit">Submit</button>
<button type="reset">Reset</button>
  • Clickable element
  • Can submit forms
  • Can trigger JavaScript events

Span

The <span> tag is an inline container used to style small parts of text inside other elements. 

<p>This is <span>important</span> text.</p>

HTML Media

HTML media elements are used to display audio, video, and images on a web page. They make websites more interactive and engaging for users.

Image

The <img> tag is used to display images. It is a self-closing tag and requires the src attribute. 

<img src="image.jpg" alt="Image description">
  • src → Image source
  • alt → Alternative text for accessibility
  • width / height → Image size

Audio

The <audio> tag is used to embed sound content such as music or voice. 

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support audio.
</audio>
  • controls → Play, pause, volume
  • autoplay → Plays automatically
  • loop → Repeats audio

Audio Output

Video

The <video> tag is used to embed video content on a web page. 

<video controls width="300">
  <source src="video.mp4" type="video/mp4">
  Your browser does not support video.
</video>
  • controls → Video controls
  • autoplay → Auto play video
  • loop → Repeats video
  • muted → Mutes audio

Video Output

Multiple Media Sources

Multiple source files can be provided for better browser support. 

<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
</video>

Embedded Media

External media like YouTube videos can be embedded using iframes. 

<iframe
  width="300"
  height="200"
  src="https://www.youtube.com/embed/VIDEO_ID">
</iframe>

Common Media Attributes

  • controls → Show controls
  • autoplay → Auto play media
  • loop → Repeat media
  • muted → Mute sound
  • poster → Thumbnail image for video

Best Practices for Media

  • Use optimized file sizes
  • Always provide alt text for images
  • Use controls for better UX
  • Avoid autoplay with sound

Favicon

A favicon (short for favorite icon) is a small icon that appears in the browser tab, bookmarks, and address bar of a website.

Why Favicon is Important?

  • Improves website branding
  • Makes site easily recognizable
  • Professional look

How to Add Favicon

Favicons are added inside the <head> section of an HTML document. 

<link rel="icon" href="favicon.ico">

Favicon with Image Formats

Different image formats can be used for favicons. 

<link rel="icon" type="image/png" href="favicon.png">
<link rel="icon" type="image/svg+xml" href="favicon.svg">

Recommended Sizes

  • 16×16 → Browser tabs
  • 32×32 → Taskbar & shortcuts
  • 48×48 → Desktop icons

Favicon Example (Complete)

<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
  <link rel="icon" href="favicon.ico">
</head>
<body>
  <h1>Welcome</h1>
</body>
</html>

Best Practices

  • Use simple and clear icons
  • Keep file size small
  • Place favicon in root folder
  • Use .ico or .png format

HTML Graphics

HTML graphics are used to draw shapes, icons, charts, and animations directly inside the browser without using images.

SVG (Scalable Vector Graphics)

SVG is an XML-based vector image format used to draw scalable graphics. SVG graphics do not lose quality when resized. 

<svg width="200" height="100">
  <rect width="200" height="100" fill="blue" />
</svg>
  • Scales without losing quality
  • SEO friendly
  • Can be styled with CSS

SVG Shapes

<svg width="200" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
  <line x1="0" y1="0" x2="200" y2="100" stroke="black" />
</svg>

Canvas

The <canvas> element is used to draw graphics using JavaScript. It is mainly used for games, animations, and dynamic graphics. 

<canvas id="myCanvas" width="200" height="100">
Your browser does not support canvas.
</canvas>
  • Bitmap based
  • Requires JavaScript
  • Used for animations and games

SVG vs Canvas

  • SVG → Vector, scalable, CSS-friendly
  • Canvas → Pixel-based, JavaScript driven

HTML APIs

HTML APIs allow web applications to interact with browser features like storage, location, and multimedia.

Geolocation API

Used to get the user's current location. 

<button onclick="getLocation()">Get Location</button>

<script>
function getLocation() {
  navigator.geolocation.getCurrentPosition(showPosition);
}
function showPosition(position) {
  console.log(position.coords.latitude, position.coords.longitude);
}
</script>

Local Storage API

Local Storage stores data in the browser even after the page is refreshed. 

<script>
localStorage.setItem("name", "Hifza");
let user = localStorage.getItem("name");
</script>
  • Stores key-value pairs
  • Data persists after reload
  • Maximum ~5MB storage

Session Storage API

Session Storage stores data temporarily for one browser session. 

<script>
sessionStorage.setItem("token", "12345");
</script>

Drag and Drop API

Allows users to drag elements and drop them into another area. 

<div draggable="true">Drag me</div>

HTML API Examples

  • Geolocation API
  • Web Storage API
  • Drag and Drop API
  • Media API

Use Cases

  • Maps & location-based apps
  • Saving user preferences
  • Interactive web applications

HTML Iframes

The <iframe> (inline frame) tag is used to embed another web page, video, map, or document inside the current HTML page.

Basic Iframe Example

<iframe src="https://example.com"></iframe>
  • Embeds another webpage
  • Works like a window inside a page

Iframe with Width and Height

<iframe src="https://example.com"
width="300"
height="200"></iframe>

Iframe with Border Control

<iframe src="https://example.com"
style="border:2px solid black;"></iframe>

Embedding YouTube Video

<iframe
width="300"
height="200"
src="https://www.youtube.com/embed/VIDEO_ID"
allowfullscreen>
</iframe>

Embedding Google Map

<iframe
src="https://www.google.com/maps"
width="300"
height="200">
</iframe>

Iframe Attributes

  • src → URL of embedded page
  • width → Width of iframe
  • height → Height of iframe
  • title → Accessibility support
  • allowfullscreen → Fullscreen mode

Iframe with Title (Recommended)

<iframe
src="https://example.com"
title="Example Website">
</iframe>

Security Considerations

  • Some websites block iframe embedding
  • Use trusted sources only
  • Avoid embedding unsecure content

When to Use Iframes

  • Embedding videos
  • Maps integration
  • External widgets

When to Avoid Iframes

  • For main page content
  • SEO critical information
